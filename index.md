Project Aura: Intelligence for the Modern Orchard

​Welcome to the technical core of Powerdreams. Project Aura is an open-source initiative dedicated to bridging the gap between Edge AI and Sustainable Agriculture, specifically focused on the unique challenges of Avocado farming in Juja, Kenya.

The Vision

​Traditional farming relies on intuition and history. Project Aura replaces guesswork with precision. By leveraging the Raspberry Pi 5 and custom sensor arrays, we monitor the micro-climates of individual Hass and Fuerte seedlings to optimize growth, reduce water waste, and predict nutrient needs before deficiencies appear.

Current Focus: The April 2026 Planting Cycle

​As we enter the long rains, our primary technical objectives are:
​Precision Irrigation: Using soil moisture logic to prevent root rot during heavy rains.
​Nutrient Tracking: Automating the "Nutrient Roadmap" via local Python APIs.
​Environmental Resilience: Monitoring humidity and temperature to predict fungal risks.

The Tech Stack

​Hardware: Raspberry Pi 5 (8GB), NPK Soil Sensors, DHT22 Temperature/Humidity sensors.
​Backend: Python 3.12, FastAPI (Local Prediction API).
​Infrastructure: Ngrok Tunnels for live data relay to Powerdreams.top.
​Documentation: Material for MkDocs hosted on GitHub Pages.
